# Nine-Security 網站維護手冊 (Website Maintenance Guide)

這份文件旨在協助您快速理解網站結構，並指導如何更新內容、新增文章或修改設計。

---

## 1. 檔案結構 (File Structure)

您的網站位於 `Security-Website` 資料夾中，主要結構如下：

```text
Security-Website/
├── assets/              # 存放圖片資源
│   ├── logo.png         # 舊版 Logo
│   └── logo_v2.png      # 目前使用的 Logo (新版)
├── index.html           # 網站主頁 (所有內容都在這裡)
├── style.css            # 網站樣式表 (控制顏色、排版、特效)
├── script.js            # 互動腳本 (背景粒子動畫)
└── WEBSITE_MAINTENANCE.md # 本教學文件
```

---

## 2. 如何更新網站內容？

所有的文字內容都位於 `index.html` 檔案中。您可以使用任何文字編輯器 (如 VS Code, Cursor, Notepad++) 開啟並編輯。

### 🟢 修改 Logo
目前 Logo 使用的是 `assets/logo_v2.png`。
- **更換圖片**：直接將新圖片命名為 `logo_v2.png` 並覆蓋 `assets/` 資料夾中的同名檔案即可。
- **調整大小**：開啟 `style.css`，搜尋 `.logo-img`，修改 `height` 數值：
  ```css
  .logo-img {
      height: 55px; /* 修改這裡的數字 */
  }
  ```

### 🟢 更新「九層防護模型」文字 (9SEC Model)
這部分的內容位於 `index.html` 的 `<section id="model">` 區塊內。
每一「層」都是一個 card，結構如下：

```html
<!-- L1 -->
<div class="card layer-card" style="--layer-color: #4a90e2;">
    <div class="card-header">
        <span class="layer-num">01</span>
        <h3>Governance & Risk</h3> <!-- 層級名稱 -->
    </div>
    <div class="card-body">
        <p class="layer-desc">Building the correct...</p> <!-- 核心描述 -->
        <ul class="layer-details">
            <li>Policy & Responsibility</li> <!-- 細項 1 -->
            <li>Risk Assessment</li>         <!-- 細項 2 -->
        </ul>
    </div>
</div>
```
直接修改標籤內的文字即可更新。

### 🟢 新增/修改軍火庫工具 (Arsenal)
位於 `<section id="tools">`。
若要新增工具，複製整個 `<div class="card tool-card">...</div>`區塊，貼在現有工具的後方，然後修改內容。

```html
<div class="card tool-card">
    <div class="card-header">
        <i class="fa-solid fa-code"></i> <!-- 圖示 (可到 FontAwesome 找) -->
        <h3>Your_New_Tool</h3>
    </div>
    <div class="card-body">
        <p>工具描述...</p>
        <div class="tech-stack">
            <span>Python</span> <!-- 技術標籤 -->
        </div>
    </div>
    ...
</div>
```

### 🟢 發布新文章 (Intelligence)
位於 `<section id="articles">`。
複製一個 `<article class="list-item">` 區塊並修改：

```html
<article class="list-item">
    <div class="date">2026-02-02</div> <!-- 日期 -->
    <div class="content">
        <h3>文章標題</h3>
        <p>文章摘要...</p>
    </div>
    <div class="action">
        <a href="文章連結.html" class="read-btn">READ_ONLY</a>
    </div>
</article>
```

## 3. 調整視覺風格 (CSS)

如果您想更改網站的主題顏色，請開啟 `style.css`，在最上方的 `:root` 區域進行設定：

```css
:root {
    --bg-color: #050505;          /* 背景顏色 (深黑) */
    --accent-color: #00ff41;      /* 主要強調色 (駭客綠) */
    --accent-secondary: #00e5ff;  /* 次要強調色 (賽博藍) */
    /* ... */
}
```
只要修改這裡的顏色代碼，整個網站的配色就會自動更新。

## 4. 如何發布到 GitHub？

當您在本地端完成修改後，請開啟終端機 (Terminal) 並執行以下指令將更新推送到 GitHub Pages：

```bash
# 1. 確保在網站資料夾內
cd d:\cursor\Nine-Security.INC\Security-Website

# 2. 加入所有變更
git add .

# 3. 提交變更 (引號內為註解)
git commit -m "更新網站內容"

# 4. 推送到 GitHub
git push
```

推送後，稍等約 1-2 分鐘，您的網站 (`https://您的帳號.github.io`) 就會顯示最新內容了！
